# Soul Mirror With Other · 灵魂镜子 (多人版)

> **"在彼此的眼中，看见未曾察觉的自己。"**

`Soul Mirror With Other` 是一个基于 AI 引导的多人互动对话系统。它不只是一个提问机器，而是一面有温度的“镜子”，旨在通过结构化的心理学引导、渐进式的自我表露以及真诚的镜映，帮助伴侣、朋友或亲子在深度对话中建立更真实、更深层的心理联结。

---

## ✨ 核心哲学：AI 作为“平等的镜子”

不同于传统的、带有评判色彩的“专家”视角，本项目中的 AI 扮演着 **“有温度的引导者 + 镜头 + 游戏主持人”** 的角色。

- **不诊断、不评价**：我们关注的是对话本身流淌出来的内容，而非给玩家贴标签。
- **共情且真实**：AI 会像一位睿智且温柔的老朋友，敏锐地捕捉对话中的细微情感与联结。
- **动态递进**：遵循“乔哈里视窗”理论，通过 🌱 感知格、🌿 联结格、🌳 反思格三个层级，安全且自然地引导对话由浅入深。

---

## 🎮 互动模式

| 模式 | 适用场景 | 核心目标 |
| :--- | :--- | :--- |
| 💫 **情侣对话** | 伴侣、爱人 | 跨越日常琐碎，探索未曾展现的深层自我，建立灵魂联结。 |
| 🌿 **朋友探索** | 好友、死党 | 打破社交面具，扩大“公开区”，发现彼此眼中那个闪闪发光的对方。 |
| 🌸 **亲子对话** | 父母与子女 | 跨越代际隔阂，在平等的视角下重新认识彼此，照见真实的对方。 |

---

## 🛡️ 安全与保护机制

我们深知深度对话可能触及脆弱的边界，因此内置了严密的安全防线：

1. **红黄灯口令**：玩家可以随时喊出“黄灯”（降维到轻松话题）或“红灯/跳过”（立即停止该话题），无需任何理由。
2. **情绪降维预警**：当 AI 监测到火药味、指责或情绪过载时，会主动提醒并建议切换到更安全的互动深度，始终将“关系的安全感”置于“真相”之上。
3. **专业人员协同**：支持心理咨询师或引导者共同参与。AI 能以“第二面结构化镜子”的身份与专业人员平等协作，提供独特的观察视角。

## 🚀 如何使用

本项目是一个 **Skill (指令集)**，你可以将其导入任何支持复杂 System Prompt 的 AI 助手（如 Claude, GPT-4 等）。

### 触发词
当用户表达以下意愿时，Skill 会自动激活：
- "我想和伴侣玩个深度对话游戏"
- "帮我和朋友做一个互动"
- "我想更了解我的孩子/父母"
- "开始灵魂镜子多人游戏"

为了帮助 AI Agent (如 Claude, GPT-4, Cursor, Windsurf 等) 一次性读入所有上下文信息，建议直接使用项目根目录下的：

👉 **[SOUL_MIRROR_COMPLETE_PROMPT.md](./SOUL_MIRROR_COMPLETE_PROMPT.md)**

该文件整合了 `SKILL.md` 的所有逻辑与 `references/` 下的所有题库。你只需要将该文件的内容作为 System Prompt 喂给 AI，即可获得完整的 Skill 能力，无需手动分批导入参考文档。

---

## 🌈 结束仪式

告别不应是匆忙的。在游戏结束时，AI 会提供三种落幕仪式供玩家选择：
- **A. 专属镜映**：为每位玩家送上一段关于“我看见的你”的反馈。
- **B. 洞见拾遗**：提炼本场对话中最动人的瞬间或深刻的洞见。
- **C. 留白落幕**：简短温馨地告别，将余韵留给玩家。
